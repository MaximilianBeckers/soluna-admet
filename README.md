# Repository for predictive ADMET modelling


### Data

Data is stored in /data


### Build environment

From within the repo, run

    uv venv
    source .venv/bin/activate
    uv sync

to build the environment using uv. If you do not have uv installed yet, you can install it with

    curl -LsSf https://astral.sh/uv/install.sh | sh
